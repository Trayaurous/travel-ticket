# Travel Ticket

这是用照片制作票根的网页应用，已经部署到pages里。

## 使用方式

- 左侧为预览区，右侧是编辑面板。
- 点击预览区上方图库按钮上传图片。
- 右侧可自定义文字和背景颜色，标题栏可换行，背景/票根/字体颜色可切换，自动推荐照片中最主要的五种颜色，每种颜色各自预设五种不同的明度方便搭配。
- 可选导出比例，比例不会在预览区里直接显示，只影响导出时填充的背景像素颜色。
- 所有颜色提取、预览和 PNG 导出都在浏览器本地完成，不依赖后端。

## 文件

- `index.html`：页面结构
- `styles.css`：新拟物界面样式
- `app.js`：图片上传、主色提取、票根绘制和导出逻辑
